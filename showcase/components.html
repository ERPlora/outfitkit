Compuestos

ok-data-table Tabla de datos con búsqueda, orden, paginación, filtros, selector de columnas, vista tabla/card, export/import. Datos por property binding. Port 1:1 del antiguo <data-table> — no cambiar la API. 
// property binding en JS (NO atributos)
dt.columns = [
  { key: 'name', header: 'Producto', sortable: true, filterable: true },
  { key: 'price', header: 'Precio', align: 'right', format: (r) => `€${r.price}` },
];
dt.rows = [{ id: 1, name: 'Café', price: 1.2 }, …];
dt.searchKeys = ['name'];
dt.pageSize = 10;
dt.columnSelector = true;
dt.views = ['table', 'card'];
dt.exportable = true;
dt.actions = [{ id: 'edit', label: 'Editar', icon: 'create-outline' }];
dt.addEventListener('rowAction', (e) => console.log(e.detail.actionId, e.detail.row));
TipoNombreDetalle

proprows · searchKeys · pageSizebinding JS
