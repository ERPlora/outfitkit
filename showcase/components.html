Componentes

Los componentes ok-* que OutfitKit añade sobre Ionic. Cada bloque trae un ejemplo en vivo, su snippet y la API. Filtra con el buscador.

<ok-data-table>
Tabla de datos

Tabla rica: búsqueda, orden, paginación, selección, vistas tabla/tarjetas, export/import CSV, acciones por fila y acción primaria. Port 1:1 del antiguo <data-table>.

 
dt.columns = [{ key: 'name', header: 'Empleado' }, …];
dt.rows = […];
dt.searchKeys = ['name', 'email'];
dt.selectable = true;
dt.views = ['table', 'cards'];
dt.primaryAction = { label: 'Nuevo', icon: 'add' };
dt.actions = [{ id: 'edit', label: 'Editar', icon: 'create-outline' }];
dt.addEventListener('rowAction', (e) => …); // { actionId, row }
NombreTipoDetalle
prop .columnsDataTableColumn[]{key, header, format?, align?, sortable?, filterable?, render?, hidden?, width?}
prop .rowsobject[]Filas a mostrar
prop .searchKeysstring[]Campos sobre los que busca el searchbar
prop pageSizenumberFilas por página
prop .actionsDataTableAction[]{id, label, icon?, color?} por fila
prop .viewsstring[]['table','cards']
prop title · selectable · .rowKeystring · bool · fn|stringTítulo, selección por fila, clave estable
prop columnSelector · exportable · importableboolSelector de columnas / export / import CSV
prop .primaryAction{label, icon?}Botón destacado de la topbar
prop .cardTitle · .cardIcon · .renderCardfnRender de la vista «cards»
evt rowAction{actionId, row}Acción de fila pulsada
evt primaryAction · selectionChange· {keys}Acción primaria · cambio de selección
evt pageChange · sortChange · searchChange · filterChange · viewChangevariosEventos de interacción
slot toolbar—Controles extra en la topbar
<ok-tree>
Árbol

Árbol expandible por datos, render recursivo con indentación por nivel. Ionic no trae un tree.

 
tree.nodes = [{ id, label, icon?, children?, expanded? }];
tree.selectable = true;
tree.activeId = 'b';
tree.addEventListener('ok-select', (e) => …); // { id, node }
NombreTipoDetalle
prop .nodesOkTreeNode[]{id, label, icon?, children?, expanded?, disabled?}
prop selectableboolFilas clicables / resaltado
prop active-idstringNodo resaltado
evt ok-toggle{id, expanded}Expandir/colapsar
evt ok-select{id, node}Selección de nodo
<ok-inline-feedback>
Feedback inline

Banner/callout persistente en el flujo del contenido (Ionic solo trae toast/alert efímeros).
Tu plan incluye 3 módulos activos. Los cambios se guardaron correctamente. Stock bajo en 3 artículos. Ver No se pudo conectar con el Hub. 
<ok-inline-feedback tone="warning" heading="Atención" dismissible>
  Stock bajo en 3 artículos.
  <ion-button slot="actions" size="small" fill="outline">Ver</ion-button>
</ok-inline-feedback>
NombreTipoDetalle
prop toneinfo|success|warning|danger|neutralColor e icono por defecto
prop heading · icon · dismissiblestring · string · boolTítulo, icono override, botón cerrar
slot (default) · actions—Mensaje · botones de acción
evt ok-dismiss—Al cerrar
<ok-empty-state>
Estado vacío

Estado centrado para «sin datos / sin resultados», con icono, título, mensaje y acción.
Crear pedido 
<ok-empty-state icon="cart-outline" heading="Sin pedidos"
  message="Cuando llegue tu primer pedido aparecerá aquí.">
  <ion-button slot="action" size="small">Crear pedido</ion-button>
</ok-empty-state>
NombreTipoDetalle
prop icon · heading · messagestringIcono, título, mensaje muted
slot (default) · action—Contenido extra · acción
<ok-kpi>
KPI

Tarjeta de métrica para dashboards: label, valor grande y delta con flecha y color según tendencia.

 
<ok-kpi label="Ventas hoy" value="€2.480" delta="+12%" trend="up" icon="cash-outline"></ok-kpi>
NombreTipoDetalle
prop label · value · deltastringEtiqueta, valor, variación
prop trendup|down|flatColor y flecha del delta
prop iconstringion-icon junto al label
slot (default)—Contenido extra (p.ej. sparkline)
<ok-stat>
Stat

Métrica inline compacta (más ligera que ok-kpi): label, valor y hint.

 
<ok-stat label="Empleados" value="6" hint="2 inactivos"></ok-stat>
NombreTipoDetalle
prop label · value · hintstringEtiqueta, valor, texto secundario
<ok-stepper>
Stepper

Indicador de pasos: círculos numerados conectados; completado / activo / pendiente. Compacto en móvil.

 
stepper.steps = [{ label, description? }, …];
stepper.current = 1;
stepper.addEventListener('ok-step-select', (e) => …); // { index }
NombreTipoDetalle
prop .stepsOkStep[]{label, description?}
prop currentnumberÍndice (0-based) activo
evt ok-step-select{index}Al pulsar un paso
<ok-wizard>
Wizard

Asistente multi-paso: stepper + contenido del paso por slots step-0, step-1… + navegación Atrás/Siguiente/Finalizar.

Paso 1 — datos del negocio.

Paso 2 — elige módulos.

Paso 3 — confirma y termina.

 
wizard.steps = [{ label: 'Negocio' }, { label: 'Módulos' }, { label: 'Listo' }];
<ok-wizard>
  <div slot="step-0">…</div>
  <div slot="step-1">…</div>
</ok-wizard>
wizard.addEventListener('ok-finish', () => …);
NombreTipoDetalle
prop .steps · currentOkStep[] · numberPasos y paso activo
prop backLabel · nextLabel · finishLabelstringTextos de los botones
slot step-0, step-1, …—Contenido de cada paso
evt ok-step-change · ok-finish{index} · —Cambio de paso · finalizar

form.addEventListener('ok-submit', (e) => …); // { name, email, subject, message }
