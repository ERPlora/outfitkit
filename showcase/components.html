Componentes

Los componentes ok-* que OutfitKit añade sobre Ionic. Cada bloque trae un ejemplo en vivo, su snippet y la API. Filtra con el buscador.

<ok-data-table>
Tabla de datos

Tabla rica: búsqueda, orden, paginación, selección, vistas tabla/tarjetas, export/import CSV, acciones por fila y acción primaria. Port 1:1 del antiguo <data-table>.

 
dt.columns = [{ key: 'name', header: 'Empleado' }, …];
dt.rows = […];
dt.searchKeys = ['name', 'email'];
dt.selectable = true;
dt.views = ['table', 'cards'];
dt.primaryAction = { label: 'Nuevo', icon: 'add' };
dt.actions = [{ id: 'edit', label: 'Editar', icon: 'create-outline' }];
dt.addEventListener('rowAction', (e) => …); // { actionId, row }
NombreTipoDetalle
prop .columnsDataTableColumn[]{key, header, format?, align?, sortable?, filterable?, render?, hidden?, width?}
prop .rowsobject[]Filas a mostrar
prop .searchKeysstring[]Campos sobre los que busca el searchbar
prop pageSizenumberFilas por página
prop .actionsDataTableAction[]{id, label, icon?, color?} por fila
prop .viewsstring[]['table','cards']
prop title · selectable · .rowKeystring · bool · fn|stringTítulo, selección por fila, clave estable
prop columnSelector · exportable · importableboolSelector de columnas / export / import CSV
prop .primaryAction{label, icon?}Botón destacado de la topbar
prop .cardTitle · .cardIcon · .renderCardfnRender de la vista «cards»
evt rowAction{actionId, row}Acción de fila pulsada
evt primaryAction · selectionChange· {keys}Acción primaria · cambio de selección
evt pageChange · sortChange · searchChange · filterChange · viewChangevariosEventos de interacción
slot toolbar—Controles extra en la topbar

Banner/callout persistente en el flujo del contenido (Ionic solo trae toast/alert efímeros).

</ok-inline-feedback>
NombreTipoDetalle
prop toneinfo|success|warning|danger|neutralColor e icono por defecto
prop heading · icon · dismissiblestring · string · boolTítulo, icono override, botón cerrar
slot (default) · actions—Mensaje · botones de acción
evt ok-dismiss—Al cerrar

Tarjeta de métrica para dashboards: label, valor grande y delta con flecha y color según tendencia.

Paso 3 — confirma y termina.

 
wizard.steps = [{ label: 'Negocio' }, { label: 'Módulos' }, { label: 'Listo' }];
<ok-wizard>
  <div slot="step-0">…</div>
  <div slot="step-1">…</div>
</ok-wizard>
wizard.addEventListener('ok-finish', () => …);

Workflow

<ok-calendar>
Calendario

Calendario mensual / agenda con eventos por día, navegación de mes y selección de fecha. Ionic no trae calendario.

 
cal.events = [{ id, date: '2026-06-12', title: 'Inventario', color: 'success' }];
cal.value = '2026-06-12';
cal.view = 'month'; // 'month' | 'agenda'
cal.maxPerDay = 3;
cal.addEventListener('ok-date-select', (e) => …); // { date }
cal.addEventListener('ok-event-click', (e) => …); // { id, event }
NombreTipoDetalle
prop .eventsOkCalendarEvent[]{id, date:'YYYY-MM-DD', title, color?}
prop valuestringFecha seleccionada (YYYY-MM-DD)
prop viewmonth|agendaModo de vista
prop max-per-daynumberMáx. eventos visibles por celda
evt ok-date-select{date}Selección de día
evt ok-event-click{id, event}Click en un evento
evt ok-view-change · ok-nav{view} · {year, month}Cambio de vista · navegación de mes

Tablero de columnas con tarjetas arrastrables entre columnas (drag & drop). Útil para pipelines y estados.

    { id: '1', title: 'Pedir stock', subtitle: 'Almacén', tags: ['urgente'] },
  ]},
  { id: 'doing', title: 'En curso', cards: [] },
];
kanban.addEventListener('ok-card-move', (e) => …); // { cardId, fromColumn, toColumn, toIndex }
kanban.addEventListener('ok-card-click', (e) => …); // { id, card }
NombreTipoDetalle
prop .columnsOkKanbanColumn[]{id, title, color?, cards:[{id, title, subtitle?, tags?}]}
evt ok-card-move{cardId, fromColumn, toColumn, toIndex}Tarjeta movida
evt ok-card-click{id, card}Click en tarjeta

Combobox

Selector con búsqueda (autocomplete): escribe para filtrar opciones y elige una.

combo.value = 'es';
combo.addEventListener('ok-input', (e) => …);   // { query }
combo.addEventListener('ok-change', (e) => …);  // { value, label }

tags.addEventListener('ok-change', (e) => …); // { tags }

evt ok-change{tags}Cambio de etiquetas
