¿Qué es OutfitKit?

OutfitKit es una librería de Web Components (Lit) que construye lo que Ionic NO tiene —tabla de datos rica, árbol, KPIs y métricas, stepper/wizard, feedback inline, estados vacíos, formulario de contacto y chrome web (navbar/footer/hero)— sobre Ionic. Ionic es la base: para botones, inputs, listas y layout usas Ionic directo (como este propio panel). OutfitKit cubre los huecos con un set ok-* tematizado por tokens --ok-*.
